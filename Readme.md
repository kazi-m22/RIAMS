# Remote Industry Appliance Management System

An IoT arrangement to control industry appliances: motor, generator etc. using  a  web  tool with **live data recording**

## Software tools:
  - Adafruit IO
  - Python

Hardware tools:
  a.DC motor
  b.Servo motor 
  c.Hall sensor
  d.Thermistor
  e.MOSFET(n type)
  f.Registorg.LED
  h.Arduino
  i.**Raspberry Pi**
